Sitio web moderno para tienda de ropa

Objetivo: Crear un sitio web moderno y minimalista para una tienda de ropa online.

Tareas realizadas:

  • Diseño UI/UX adaptado a la marca.
  • Maquetación HTML/CSS y Bootstrap responsive.
  • Implementación de catálogo de productos y carrito de compras.

Resultados: Incremento de visitas y ventas online, sitio con diseño moderno y funcional.

Campaña digital para emprendimiento de comida saludable

Objetivo: Promocionar un emprendimiento de comida saludable mediante campañas digitales.

Tareas realizadas:

  • Diseño de imágenes y banners para redes sociales.
  • Planificación de publicaciones y anuncios.
  • Optimización de contenido visual y textual para engagement.

Resultados: Aumento de interacciones y alcance en redes sociales, mayor reconocimiento de marca.

Gestión de redes sociales para Dlass

  • Planificación de calendario de publicaciones.

  • Monitoreo y análisis de métricas de engagement.

Desarrollo de landing page para curso online

Objetivo: Crear una landing page atractiva y funcional para promocionar un curso online.

Tareas realizadas:

  • Diseño visual y estructuración de la landing page.
  • Implementación de formulario de inscripción y CTA claros.
  • Optimización responsive y velocidad de carga.

Resultados: Mayor conversión de registros y experiencia de usuario mejorada.

Rediseño web para emprendimiento de artesanías

Objetivo: Modernizar el sitio web de un emprendimiento de artesanías para mejorar su presencia online.

Tareas realizadas:

  • Rediseño UI/UX adaptado a la identidad de marca.
  • Maquetación responsive y optimización de imágenes.
  • Integración de galería de productos y tienda online.

Resultados: Sitio más atractivo, mayor tiempo de visita y aumento de pedidos online.

Divina Energía

Objetivo: Potenciar la presencia online de Divina Energía, un emprendimiento de terapias energéticas, generando contenido visual atractivo y educativo.

Tareas realizadas:

  • Creación de calendario de publicaciones y planificación estratégica semanal.
  • Diseño de imágenes y reels explicativos sobre beneficios de Reiki y meditación.
  • Redacción de textos educativos y persuasivos para Instagram y TikTok.
  • Optimización de hashtags y horarios de publicación para mayor visibilidad.
  • Interacción con la comunidad mediante encuestas y contenido interactivo.

Resultados: Incremento de engagement, mayor visibilidad de la marca y generación de clientes potenciales.
